# Employee Management System (React)

This project is a simple **Employee Management Dashboard** built using **React.js**. The application provides separate interfaces for **Admin and Employees**, allowing task assignment and management in an organized way.

## 🚀 Features

* Admin login and Employee login system
* Admin dashboard to assign tasks to employees
* Employee dashboard to view assigned tasks
* Task status management (New Task, Active Task, Completed Task)
* Data persistence using **LocalStorage**
* Responsive and modern UI built with **React and Tailwind CSS**

## 🛠 Tech Stack

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* HTML5
* LocalStorage (for data persistence)

## 📂 Project Functionality

The application simulates a basic company workflow:

* **Admin Panel**

  * Assign tasks to employees
  * Track task status
  * Manage employee activities

* **Employee Panel**

  * View assigned tasks
  * Update task progress
  * Mark tasks as completed

All user data and tasks are stored in **localStorage**, allowing the application to maintain state even after refreshing the page.

## 🎯 Purpose of the Project

This project is built to practice **React fundamentals**, including:

* Component-based architecture
* Props and state management
* Form handling
* Conditional rendering
* LocalStorage integration

It is a beginner-friendly project for developers learning **React and frontend application development**.
